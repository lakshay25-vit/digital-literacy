# Digital Literacy Project

## Project Overview
This project is part of the Digital Literacy module and focuses on developing essential digital skills required in today's professional and academic environments. The project covers multiple aspects of digital competency, including visual communication, professional online presence, basic programming, email etiquette, collaboration tools, and cybersecurity awareness.

Through these modules, students learn how to effectively use modern digital tools while maintaining responsible and secure online behavior.

## Module Summaries

### Module 1 – Infographic Creation
This module focuses on creating informative and visually appealing infographics using tools such as **Prezi** and **Canva**. It teaches how to present information in a clear, engaging, and creative format to communicate ideas effectively.

### Module 2 – Building Social Presence
This module explains how to improve professional visibility online through platforms such as **LinkedIn** and other professional profiles. It highlights the importance of networking, maintaining a professional digital identity, and showcasing skills and achievements online.

### Module 3 – Basic Coding Skills
This module introduces the fundamentals of programming through coding practice platforms like **CodeChef** and **HackerRank**. It helps students develop problem-solving skills and basic coding knowledge through practical exercises.

### Module 4 – Email Etiquette and Collaboration Tools
This module focuses on professional communication through proper **email etiquette** and the effective use of collaboration platforms such as **Google Workspace**. It teaches students how to communicate clearly, maintain professionalism, and collaborate efficiently in digital environments.

### Module 5 – Cybercrime Awareness
This module covers the concept of **cybercrime**, including common online threats such as phishing, identity theft, and hacking. It also discusses preventive measures and best practices to stay safe in the digital world.
